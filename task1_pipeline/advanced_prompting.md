# Advanced Prompt Engineering

## Objective
Study how prompt wording influences Stable Diffusion image generation.

## Experiments
- Realistic prompts
- Anime prompts
- Cinematic prompts
- Style prompts
- Negative prompts

## Key Learnings
- Prompt structure strongly affects output quality
- Style tokens influence lighting and aesthetics
- Negative prompts reduce unwanted artifacts
- Detailed prompts improve semantic consistency
